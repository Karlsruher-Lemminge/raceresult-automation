# Feld-Auswahl #

Die **Feld-Auswahl** erscheint an zahlreichen Stellen in der RACE RESULT Software unter Einstellungsfeldern, z.B. in den Einstellungen von [[@1120|Listen]] oder bei der Definition von [[@1151|Platzierungen]]. Sie ermöglicht die einfache Auswahl von [[@2757|Feldern]], die z.B. auf einer Liste angezeigt werden sollen.
